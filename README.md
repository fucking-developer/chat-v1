# Chat en Tiempo Real con WebSockets

## 📌 Descripción
Este proyecto es un chat en tiempo real basado en **Node.js**, **Express** y **WebSockets**, donde los usuarios pueden enviar preguntas y un administrador puede gestionarlas desde una interfaz de administración.

## 🚀 Instalación y Uso
### 1️⃣ Clonar el repositorio
```sh
 git clone https://github.com/fucking-developer/chat-v1.git
 cd mi-proyecto-chat
```

### 2️⃣ Instalar dependencias
```sh
 npm install express ws body-parser
```

### 3️⃣ Ejecutar el servidor
```sh
 node server.js
```

### 4️⃣ Acceder al chat
- **Usuarios:** `http://localhost:8080`
- **Iniciar sesión:** `http://localhost:8080/login`
- **Administrador:** `http://localhost:8080/admin`

### 5️⃣ Credenciales de acceso (Admin)
- **Usuario:** `admin`
- **Contraseña:** `1234`
